# Repository Tree Validation Spec

The repository root must include and preserve the following baseline paths:

- `apps/`
- `packages/`
- `docs/`
- `tools/`
- `artifacts/`
- `specs/`
- `README.md`
- `CONTRIBUTING.md`
- `CODE_OF_CONDUCT.md`
- `LICENSE`
- `agents.md`

Automated validator: `tools/governance/validate-tree.mjs`.

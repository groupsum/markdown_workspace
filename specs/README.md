# Specifications

This directory centralizes repository governance requirements used by automated validation tests and CI jobs.

## Included specs

- `repository-tree-spec.md`
- `doc-pointer-spec.md`
- `root-clutter-spec.md`
- `generated-artifact-protection-spec.md`
- `claim-language-spec.md`
- `wip-notes-spec.md`
- `release-note-spec.md`

Each validator prints the spec path on failure so maintainers can quickly navigate to the governing requirement.

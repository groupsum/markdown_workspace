# packages/shared

App-agnostic reusable primitives shared across MdWrk apps and packages.

## Implemented packages

- `mdwrk/ui-tokens` — shared root variables, markdown styles, class names, and theme mapping helpers
- `mdwrk/icons` — shared icon identifiers and metadata
- `mdwrk/i18n` — shared message descriptors and locale registry helpers
- `mdwrk/testing` — shared React and browser test helpers

These packages are publishable npm packages. Their `dist/` output is generated from source and is not committed.

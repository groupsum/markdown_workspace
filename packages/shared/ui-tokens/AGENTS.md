# UI Tokens And Styles Agent Instructions

This package is styles-focused. It owns shared token and CSS foundations used by UIX packages and apps.

- Use the `styles_specialist` custom agent for delegated token, CSS, or visual-parity work.
- Preserve stable token names unless a migration updates all consumers and docs.
- Keep tokens semantic and reusable; do not encode one app screen's layout as a shared token.
- When changing visible tokens, capture screenshots for all affected themes, views, modes, panes, and modals.

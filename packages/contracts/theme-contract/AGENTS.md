# Theme Contract Agent Instructions

This package is styles-focused and contract-focused. It defines the theme contract that apps and packages consume.

- Use the `styles_specialist` custom agent for delegated theme contract work.
- Keep viewport and aspect-ratio breakpoint semantics centralized in the client base viewport contract; do not redefine band logic inside individual themes.
- Update contract docs and compatibility helpers with any public token, class, or theme shape changes.
- Treat visual parity evidence as required when changing theme contract behavior.

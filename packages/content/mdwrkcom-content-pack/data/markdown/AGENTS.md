# Markdown Content Agent Instructions

Markdown data content is docs, content, and copy focused. Treat frontmatter, content body, sitemap registration, and generated static output as one publishing contract.

- Use the `content_seo_specialist` custom agent for delegated markdown content work.
- Keep frontmatter metadata specific to the page: title, description, excerpt, image, tags, and route intent should match the body.
- Preserve present-tense product copy and practical, current-state guidance.
- Update content indexes or sitemap entries whenever a page is added, moved, renamed, or removed.

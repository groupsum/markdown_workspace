# Renderer basic example

[![Hits](https://visitor-badge.laobi.icu/badge?page_id=groupsum.markdown_workspace.examples.renderer_basic.readme&left_text=hits)](https://github.com/groupsum/markdown_workspace/blob/master/examples/renderer-basic/README.md)
[![Downloads](https://img.shields.io/github/downloads/groupsum/markdown_workspace/total?label=downloads)](https://github.com/groupsum/markdown_workspace/releases)

This example validates the public `@mdwrk/markdown-renderer-react` surface.

It demonstrates:

- CommonMark core rendering
- GFM tables, task lists, strikethrough, and autolink literals
- optional extension rendering for definition lists, math, and footnotes
- theme support through `createMarkdownRendererThemeStyle()`
- runtime profile toggles for optional extensions

Public APIs exercised:

- `MarkdownRenderer`
- `createMarkdownRendererThemeStyle()`

# Editor Basics

MdWork blends a high-density editor with a live preview surface. Use the shortcuts below to move fast.

## Switching Views
Use the view toggle to move between:
- **Editor**: Pure Markdown input.
- **Preview**: Rendered output.
- **Split**: Side-by-side editing.

## Creating Files
Use the action rail for deterministic file operations:
- **New File**
- **New Folder**
- **Rename**
- **Delete**

## Project Selection
Switch projects from the header to keep focused workspaces separate.

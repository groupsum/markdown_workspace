The mdwrk client blends `@mdwrk/markdown-editor-react` with `@mdwrk/markdown-renderer-react`. The same split surface is what this lander demo now uses.

## Switching Views
Use the view toggle to move between:
- **Editor**: Pure Markdown input.
- **Preview**: Rendered output through the shared renderer package.
- **Split**: Side-by-side editing using the split package bridge.

## Creating Files
Use the action rail for deterministic file operations:
- **New File**
- **New Folder**
- **Rename**
- **Delete**

## Project Selection
Switch projects from the header to keep focused workspaces separate.

## Editor Package Surface

The reusable editor package provides:

- controlled and uncontrolled editing modes
- keyboard shortcut and command wiring
- line-number gutter support
- theme bridge helpers for shared token-driven styling

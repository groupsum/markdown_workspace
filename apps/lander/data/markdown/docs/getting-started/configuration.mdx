# Configuration

MdWrkSpace ships with sensible defaults. These settings help you tailor the editor to your workflow.

## Theme Selection
Open **Settings → Theme** to switch between the built-in theme families. Each theme respects the shared viewport contract while applying its own material treatment.

## Workspace Defaults
- **Autosave**: Enabled by default for local-first persistence.
- **Default View**: Choose Editor, Preview, or Split to match your writing style.
- **Git Sync**: Provide a GitHub token only when you want to push or pull.

## Environment Links
If you host your own landing experience, the lander reads its navigation URLs from environment variables such as `VITE_GITHUB_REPO_URL`, `VITE_NPM_REPO_URL`, and `VITE_NPM_ESM_CDN_URL`.

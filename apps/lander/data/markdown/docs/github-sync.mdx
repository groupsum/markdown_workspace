Git-based sync is optional. The mdwrk client stays fully local until you connect a repository.

## Personal Access Tokens
Create a fine-grained personal access token in GitHub and store it in the mdwrk client settings modal.

## Push & Pull
- **Pull**: Mirror a repository into your local workspace.
- **Push**: Commit updates when you are ready.

## Offline Mode
Disconnect from the network anytime. MdWork keeps all changes locally until you choose to sync.

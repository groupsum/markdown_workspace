Back-dated note: January 22, 2026.

## Rollout summary

The mdwrk client matured from a markdown shell into a governed workspace host with:

- split editor and preview packages
- theme contract and bridge variables
- extension manifest and runtime contracts
- bundled first-party extension surfaces
- schema-backed settings, i18n, and diagnostics hooks

## User-facing outcomes

This rollout added durable client-level documentation subjects:

- workspace settings and view modes
- theme selection and theme export
- extension installation and trust semantics
- package-level editor and renderer consumption

These are the subjects the lander now prioritizes.

# Markdown Lists + Strikethrough Toolbar Update

This release focuses on fast markdown authoring and cleaner preview fidelity.

## What's new

- Added a **strikethrough** convenience control using ~~text~~ in the editor toolbar right beside bold and italic.
- Improved list rendering for plaintext markdown, including:
  - unordered list spacing and marker visibility
  - ordered list numbering and nesting styles
  - list item paragraph spacing for multi-line content

## Why this matters

Markdown authors often write lists in quick passes. The updated styling makes bullets and numbered structures easier to scan while preserving a compact, readable layout in preview mode.

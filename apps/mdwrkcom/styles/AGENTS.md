# Public Site Styles Agent Instructions

Public site styles are styles-focused. Keep route layouts, markdown rendering, syntax themes, and shared lander styling centralized and semantic.

- Use the `styles_specialist` custom agent for delegated public-site CSS work.
- Prefer semantic class hooks and centralized style modules over inline style drift.
- Preserve readable markdown content across docs, blog, compare, answers, proof, trust, and package pages.
- Verify visible style changes with screenshots for affected routes, themes, and responsive bands.
